歡迎來到lamp23333的個人網站！

我一直想擁有一個個人網站，卻常常因為申請正規域名和網絡伺服器需要長期付費而卻步，直到我遇見了GitHub Pages。

好吧，我其實痛恨自己為什麼沒有早點發現 GitHub 還有這種功能。

我是一個來自中國的高中生，熱愛科技，熱愛創新，熱愛遊戲，熱愛二次元，熱愛電音，熱愛生活。我嚮往那些年輕創作者的不凡，我想成為他們那樣的人。

我目前不具有能夠支撐我行動的技能，我不會寫程式、不會畫畫、不會做音樂、不會開發遊戲、不會管理團隊，我想付出一切代價去學，但我也和其他平凡的高中生一樣，有著繁忙的學業、激烈的競爭、外界的壓力、普通的家境。現在的我似乎只是在紙上談兵罷了，但我沒有能力，也沒有道理去幹我現在想幹的事情，因為「高考」擺在我的面前。

不過也許是為了能夠快一點達成目標，我從現在開始就做了一些準備。也許是分享慾的原因，我創建了這個網站，並把我的成果放在這裡。雖然似乎不會有什麼人來看。

我說過我什麼也不會，那麼我的成果從哪兒來的？其實是AI生成的。

雖然我也不認為AI可以替代一切創造，但是在概念性的想法直接轉換成實質性內容這方面，我只能說AI還是太好用了，但在未來我如果真的投入創作的話，我絕對不會使用它。

我想說的就這麼多，希望來到這裡的人可以諒解一下我在這個網站中展示的一切幼稚又可笑的想法。

感謝相遇。
